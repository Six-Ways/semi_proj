# SEMI_PROJ：半导体器件与制造技术交互式指南

## 项目概述

SEMI_PROJ是一个专注于半导体器件与制造技术的交互式教学网站，旨在提供全面、深入、实用的半导体知识体系。项目遵循"器件结构-工作机制-性能优化-应用场景"的逻辑链，通过交互式工具帮助学习者更好地理解半导体器件的核心原理和应用。

## 项目结构

```
semi_proj/
├── index.html              # 主页
├── css/
│   └── styles.css          # 样式文件
├── js/
│   └── main.js             # 交互功能
├── assets/
│   ├── images/             # 图片资源
│   └── data/               # 数据文件
└── chapters/
    ├── chapter4.html       # 第4章：MOSFET核心技术
    └── interactive-demo.html # 交互式组件示例
```

## 主要功能

1. **响应式设计**：适配不同设备和屏幕尺寸
2. **交互式学习**：通过可视化工具和模拟器深化理解
3. **模块化内容**：按章节组织的学习内容
4. **个性化路径**：针对不同用户群体的学习路径

## 技术栈

- HTML5：页面结构和语义化标记
- CSS3：样式和响应式设计
- JavaScript：交互功能和动态内容
- Chart.js：数据可视化
- Font Awesome：图标库

## 页面说明

### 主页 (index.html)
