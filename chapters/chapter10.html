MOSFET基本结构与工作原理

MOSFET是现代集成电路的核心器件，其基本结构由源极（Source）、漏极（Drain）、栅极（Gate）和衬底（Substrate）四部分组成。栅极通过氧化物绝缘层与沟道区隔开，通过控制栅压可以调节沟道的导电性，实现开关和放大功能。

MOSFET基本结构示意图

图10-1：NMOS晶体管的基本结构

MOSFET的工作原理基于栅极电压对沟道区表面电势的调制。当栅压变化时，沟道区的载流子浓度随之改变，从而控制源漏之间的电流。

1

MOS结构基础

基于第8章的MOS结构物理，当栅极施加正电压时，P型衬底表面会形成电子反型层，这层反型层就是MOSFET的导电沟道。

2

沟道形成

当栅极电压超过阈值电压Vth时，反型层形成，连接源极和漏极，此时源漏之间可以通过载流子导电。

3

电流控制

通过改变栅极电压可以调节沟道的导电能力（沟道电导），从而控制源漏电流的大小。漏极电压的变化会影响沟道的形状，导致不同的工作区域。

增强型MOSFET

增强型MOSFET在零栅压下没有导电沟道，需要施加正栅压（对于NMOS）或负栅压（对于PMOS）来形成沟道。这是最常用的MOSFET类型，广泛应用于数字电路中。

  • 零栅压下，源漏之间电阻极高
  • 需要施加栅压才能导通
  • 阈值电压Vth > 0（NMOS）或Vth < 0（PMOS）
  • 用于数字逻辑和开关应用

耗尽型MOSFET

耗尽型MOSFET在零栅压下就已经存在导电沟道，需要施加负栅压（对于NMOS）或正栅压（对于PMOS）来耗尽沟道，关断电流。

  • 零栅压下，源漏之间可以导电
  • 需要施加反向栅压才能关断
  • 阈值电压Vth < 0（NMOS）或Vth > 0（PMOS）
  • 用于模拟电路和特殊应用

I-V特性与工作区域

MOSFET的电流-电压（I-V）特性描述了源漏电流与栅源电压、漏源电压之间的关系。根据栅源电压和漏源电压的不同，MOSFET可以工作在三个主要区域：截止区、线性区和饱和区。

MOSFET电流方程

对于长沟道MOSFET，其I-V特性可以用以下经典方程描述：

截止区（VGS < Vth）

ID ≈ 0

当栅源电压小于阈值电压时，没有反型层形成，漏源电流几乎为零。

线性区（VGS > Vth，VDS < VGS - Vth）

ID = μnCox(W/L)[(VGS - Vth)VDS - (1/2)VDS2]

当漏源电压较小时，沟道是均匀的，电流与VDS近似线性关系，因此称为线性区。

饱和区（VGS > Vth，VDS ≥ VGS - Vth）

ID = (1/2)μnCox(W/L)(VGS - Vth)2(1 + λVDS)

当漏源电压足够大时，漏端沟道被夹断，电流不再随VDS显著增加，进入饱和区。λ是沟道长度调制系数。

主要参数说明

  • μn：电子迁移率
  • Cox：单位面积栅氧化层电容
  • W/L：宽长比，决定MOSFET的驱动能力
  • Vth：阈值电压
  • λ：沟道长度调制系数

短沟道效应

随着MOSFET尺寸的不断缩小，沟道长度接近微米或纳米量级时，经典的长沟道模型不再适用，会出现一系列短沟道效应，影响器件的性能和可靠性。

阈值电压滚降

随着沟道长度的减小，源漏结耗尽区对沟道区的影响增强，导致阈值电压随沟道长度减小而降低的现象。

  • 物理机制：源漏耗尽区电荷共享
  • 影响：关态漏电流增加
  • 缓解方法：减小衬底掺杂浓度，增加栅氧化层厚度

漏致势垒降低（DIBL）

当漏源电压增加时，漏结耗尽区扩展，降低了源区到沟道的势垒，导致更多载流子从源区注入沟道，增加漏源电流。

  • 物理机制：漏电压对源区势垒的调制
  • 影响：亚阈值摆幅恶化，关态漏电流增加
  • 缓解方法：使用浅结工艺，增加栅对沟道的控制

速度饱和

在短沟道器件中，沟道电场很强，载流子速度达到饱和值，不再随电场增加而线性增加，导致电流饱和提前。

  • 物理机制：高电场下载流子速度饱和
  • 影响：饱和电流降低，跨导减小
  • 缓解方法：使用高迁移率材料，优化沟道设计

热载流子注入（HCI）

高电场下的热载流子获得足够能量，可能注入到栅氧化层中，导致阈值电压漂移和可靠性问题。

  • 物理机制：热载流子越过势垒注入栅氧化层
  • 影响：器件老化，性能退化
  • 缓解方法：优化漏端结构，降低电场峰值

短沟道效应的影响

短沟道效应综合作用的结果是：

  • 亚阈值摆幅增大，开关性能下降
  • 关态漏电流增加，静态功耗上升
  • 电流驱动能力降低，速度性能下降
  • 器件可靠性下降，寿命缩短
  • I-V特性偏离长沟道模型，建模困难

先进MOSFET结构

为了抑制短沟道效应，继续提高器件性能，科研人员开发了多种先进的MOSFET结构，主要包括多栅器件和新沟道材料器件。

FinFET结构

FinFET结构示意图

图10-2：FinFET三维结构示意图

FinFET（鳍式场效应晶体管）是一种三维多栅器件，其沟道区域形成一个"鳍"状结构，栅极从三个方向（两侧和顶部）包围沟道，显著增强了栅极对沟道的控制能力。

FinFET的优势

  • 更好的栅控能力：多栅结构有效抑制短沟道效应
  • 更低的关态漏电流：提高器件的开关比
  • 更高的驱动电流：鳍状结构增大了有效沟道宽度
  • 更好的亚阈值特性：亚阈值摆幅接近理想值
  • 兼容传统CMOS工艺：易于集成到现有制造流程中

FinFET的关键参数

鳍高（Hfin）

鳍的高度，决定了有效沟道宽度的大小。

鳍宽（Wfin）

鳍的宽度，影响沟道的量子限制效应和短沟道抑制能力。

鳍间距（Sfin）

相邻鳍之间的距离，影响器件密度和寄生电容。

栅长（Lg）

栅极的长度，决定了器件的开关速度和短沟道效应。

全环绕栅极（GAA）晶体管

GAA纳米片晶体管结构

图10-3：GAA纳米片晶体管结构示意图

全环绕栅极（GAA）晶体管是FinFET的进一步发展，其栅极从四个方向完全包围沟道，形成纳米线或纳米片结构，具有更强的栅控能力和更好的短沟道抑制效果。

GAA的优势

  • 完美的栅控：栅极完全包围沟道，电场分布均匀
  • 极致的短沟道抑制：有效控制亚10nm沟道的短沟道效应
  • 更好的性能调节：通过调整纳米片的数量和厚度优化性能
  • 更低的功耗：进一步降低漏电流和动态功耗
  • 更好的热管理：三维结构有利于热量散发

本章小结

本章详细介绍了金属-氧化物-半导体场效应晶体管（MOSFET）的工作原理、I-V特性、短沟道效应以及先进MOSFET结构。MOSFET作为现代集成电路的核心器件，其性能直接决定了芯片的速度、功耗和集成度。

核心要点

  • MOSFET通过栅极电压控制沟道的导电性，实现开关和放大功能
  • MOSFET的I-V特性分为截止区、线性区和饱和区三个主要工作区域
  • 随着沟道长度的减小，会出现阈值电压滚降、DIBL、速度饱和等短沟道效应
  • FinFET和GAA等先进多栅结构通过增强栅控能力，有效抑制短沟道效应
  • 先进MOSFET结构是延续摩尔定律的关键技术之一
