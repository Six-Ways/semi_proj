# SEMI_PROJ 移动互联网与AI时代升级方案

## 项目概述

SEMI_PROJ是一个专注于半导体器件与工艺集成知识的交互式学习平台。为了适应移动互联网时代以及未来的AI时代，我们对项目进行了全面升级，使其不仅能在电脑上良好运行，还能在各种移动设备上提供优秀的用户体验，并融入AI技术增强学习效果。

## 技术栈

### 前端技术
- **HTML5**: 语义化标签，支持现代Web标准
- **CSS3**: 响应式设计，动画效果，暗色模式支持
- **JavaScript ES6+**: 模块化代码，现代JavaScript特性
- **Chart.js**: 数据可视化，图表展示
- **Font Awesome**: 图标库
- **Prism.js**: 代码高亮

### 移动端适配技术
- **PWA (Progressive Web App)**: 离线访问，添加到主屏幕
- **响应式设计**: 多断点适配不同屏幕尺寸
- **触摸优化**: 手势支持，触摸反馈
- **性能优化**: 懒加载，资源预加载

### AI功能增强
- **智能学习助手**: 个性化学习路径推荐
- **智能问答系统**: 半导体专业知识解答
- **参数模拟器**: 器件特性实时模拟
- **学习进度追踪**: 智能分析学习效果

## 新增文件与功能

### 1. PWA支持
- **manifest.json**: PWA配置文件，定义应用元数据

  - 学习进度追踪

### 3. 移动端优化
- **js/mobile-optimization.js**: 移动端交互优化
  - 触摸手势支持

  - 多断点适配

  - 暗色模式支持

## 主要功能特性

### 1. 移动互联网适配

#### 响应式设计
- **多断点布局**: 576px, 768px, 992px, 1200px断点设计
- **流体网格**: 自适应网格系统，适应各种屏幕尺寸
- **弹性图片**: 响应式图片，根据屏幕尺寸加载合适资源
- **触摸友好**: 增大触摸目标，优化触摸交互

#### PWA功能
- **离线访问**: Service Worker缓存关键资源
- **添加到主屏幕**: Web App Manifest支持
- **推送通知**: 学习提醒和内容更新通知
- **类原生体验**: 全屏模式，启动画面

#### 性能优化
- **懒加载**: 图片和资源按需加载
- **代码分割**: 关键资源优先加载
- **缓存策略**: 智能缓存更新机制
- **触摸优化**: 减少延迟，提高响应速度

### 2. AI时代功能增强

#### 智能学习助手
- **智能问答**: 基于规则的半导体专业知识解答
- **学习路径**: 根据用户水平推荐个性化学习路径
- **进度追踪**: 分析学习进度，识别薄弱环节
- **知识关联**: 智能推荐相关学习资源

#### 交互式模拟
- **器件参数模拟**: 实时调整参数，观察器件特性变化
- **可视化图表**: 动态生成I-V特性曲线
- **多种器件类型**: 支持MOSFET、BJT、二极管等器件模拟
- **参数对比**: 不同参数设置下的特性对比

#### 个性化体验
- **学习档案**: 记录用户学习历史和偏好
- **难度自适应**: 根据用户水平调整内容深度
- **学习提醒**: 智能推送复习提醒
- **成就系统**: 学习里程碑奖励

## 使用指南

### 1. 本地开发与测试

#### 启动本地服务器
```bash
# 使用Python
python -m http.server 8000

# 或使用Node.js
npx http-server -p 8000
```

#### 测试PWA功能
1. 在Chrome中访问 `http://localhost:8000`
2. 打开开发者工具 → Application → Manifest
3. 检查Service Worker状态
4. 测试离线功能：在Network标签中选择"Offline"

#### 测试移动端体验
1. 在Chrome开发者工具中切换到移动设备视图
2. 测试不同屏幕尺寸下的响应式效果
3. 测试触摸手势和交互
